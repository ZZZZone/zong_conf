## 安装

**如果你有自己的配置，执行命令前记得备份你的配置。**

1. 一键安装  
   > 使用[dotbot](https://github.com/anishathalye/dotbot)管理。

   ```bash  
   git clone https://github.com/ZZZZone/zong_conf.git && cd zong_conf
   sh ./install
   ```
